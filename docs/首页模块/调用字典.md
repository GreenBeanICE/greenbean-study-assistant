# 首页与工作区模块 — 调用字典

> 本文档列出首页及工作区模块所有对外暴露的组件接口、Prop 类型、State 结构和 Reducer Action，供其他页面或功能模块调用时参考。

---

## 一、首页组件接口

### 1.1 SplashScreen（启动开场屏）

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| onSkip | `() => void` | 否 | — | 用户跳过/自动结束开场动画后的回调 |
| className | `string` | 否 | `""` | 附加到根元素的 CSS class |

```tsx
// 调用方示例（App.tsx）
import SplashScreen from "./features/home/components/SplashScreen";

<SplashScreen onSkip={() => setShowSplash(false)} />
```

**内部逻辑：**
1. 打开后显示占位圆点（3 个大渐变圆 + 3 个小实心点）
2. 800ms 后显示「欢迎」文字
3. 1700ms 后显示副标题「GreenBean Study Assistant」
4. 3800ms 后自动触发 `onSkip`

## 二、工作区组件接口

### 2.1 WorkspacePage（工作区主页面）

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| _props | `WorkspacePageProps` | 否 | — | 当前无必填 Prop，类型仅用于后续扩展 |

**内部状态（workspaceReducer）：**

```tsx
interface WorkspaceState {
  sections: SectionNode[];          // 章节树
  selectedSectionId: string | null; // 当前选中章节 ID
  contentBlocks: ContentBlock[];    // 文档内容块列表
  chatMessages: ChatMessage[];      // 聊天消息列表
  chatInput: string;                // 聊天输入框内容
  loading: boolean;                 // 加载状态
  footnotes: FootnoteReference[];   // 脚注引用列表
  expandedFootnoteId: string | null; // 展开的脚注 ID
  currentSelection: SelectionType | null; // 当前选中的文字
  showSelectionMenu: boolean;       // 是否显示选中菜单
  selectionMenuPos: Position | null; // 选中菜单位置
  quotedText: string | null;        // 引用的文字
  tokenUsage: number;               // Token 使用量
  leftCollapsed: boolean;           // 左侧面板是否折叠
  rightCollapsed: boolean;          // 右侧面板是否折叠
  leftPanelWidth: number;           // 左侧面板宽度（160-302px）
  rightPanelWidth: number;          // 右侧面板宽度（189-340px）
  documentTitle: string;            // 文档标题
}
```

**Reducer Action 类型：**

| Action | 负载 | 说明 |
|--------|------|------|
| `SELECT_SECTION` | `sectionId: string` | 选中章节 |
| `TOGGLE_SECTION_EXPAND` | `sectionId: string` | 展开/折叠章节 |
| `TOGGLE_HIGHLIGHT` | `blockId, lineId` | 切换高亮 |
| `UPDATE_LINE_TEXT` | `blockId, lineId, text` | 更新行文本 |
| `FORMAT_LINE` | `blockId, lineId, format` | 格式化文本（bold/italic/underline/strikethrough/highlight/align） |
| `SET_LANG_DATA` | `sections, contentBlocks` | 设置章节内容和数据 |
| `TOGGLE_FOOTNOTE` | `footnoteId` | 展开/收起脚注 |
| `SET_SELECTION` | `selection` | 设置当前选中 |
| `SHOW_SELECTION_MENU` | `show, pos?` | 显示/隐藏选中菜单 |
| `QUOTE_SELECTION` | — | 引用当前选中 |
| `CLEAR_QUOTE` | — | 清除引用 |
| `SET_CHAT_INPUT` | `text` | 更新聊天输入 |
| `SEND_CHAT_MESSAGE` | `message` | 发送聊天消息 |
| `SET_LOADING` | `loading` | 设置加载状态 |
| `SET_TOKEN_USAGE` | `usage` | 设置 Token 用量 |
| `TOGGLE_LEFT_PANEL` | — | 切换左侧面板 |
| `TOGGLE_RIGHT_PANEL` | — | 切换右侧面板 |
| `SET_LEFT_WIDTH` | `width` | 设置左侧面板宽度 |
| `SET_RIGHT_WIDTH` | `width` | 设置右侧面板宽度 |
| `SET_DOC_TITLE` | `title` | 设置文档标题 |

### 2.2 FileManager（文件管理面板）

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| files | `FileItem[]` | 否 | 内置示例文件 | 外部传入的文件列表 |
| folders | `Folder[]` | 否 | 课程资料/考试复习/论文参考 | 文件夹列表 |
| selectedFileId | `string \| null` | 否 | — | 当前选中的文件 ID |
| onFileSelect | `(fileId: string) => void` | 否 | — | 选中文件回调 |
| onFileSelectWithName | `(fileId: string, fileName: string) => void` | 否 | — | 选中文件回调（含文件名） |

```tsx
// 使用示例
import FileManager from "../components/left/FileManager";

<FileManager onFileSelectWithName={(id, name) => handleFileSelect(id, name)} />
```

**内部数据模型：**

```tsx
interface FileItem {
  id: string;
  name: string;
  type: FileType;       // "PDF" | "DOC" | "PPT" | "IMG" | "TXT" | "MD"
  category: string;     // "course" | "exam" | "thesis" | ""
  size: string;         // 如 "12.4 MB"
  date: string;         // "2025-09-15"
  status: FileStatus;   // "parsed" | "parsing" | "pending"
}

interface Folder {
  key: string;
  label: string;
}
```

**支持的操作：**
- 上传新文件（文件选择器）
- 搜索文件名
- 文件夹展开/折叠
- 右键菜单：重命名、移动到其他文件夹、移出文件夹、删除

### 2.4 DocumentViewer（文档查看器）

| Prop | 类型 | 必填 | 说明 |
|------|------|------|------|
| contentBlocks | `ContentBlock[]` | 是 | 文档内容块 |
| selectedSectionId | `string \| null` | 是 | 当前选中章节 |
| footnotes | `FootnoteReference[]` | 是 | 脚注引用列表 |
| expandedFootnoteId | `string \| null` | 是 | 展开的脚注 |
| currentSelection | `SelectionType \| null` | 是 | 当前选中 |
| showSelectionMenu | `boolean` | 是 | 是否显示选中菜单 |
| selectionMenuPos | `Position \| null` | 是 | 选中菜单位置 |
| onToggleHighlight | `(blockId, lineId) => void` | 是 | 切换高亮 |
| onUpdateLineText | `(blockId, lineId, text) => void` | 是 | 更新行文本 |
| onFormatLine | `(blockId, lineId, format) => void` | 是 | 格式化行文本 |
| onToggleFootnote | `(footnoteId) => void` | 是 | 切换脚注 |
| onSelectText | `(selection) => void` | 是 | 选中文字 |
| onShowSelectionMenu | `(show, pos?) => void` | 是 | 显示选中菜单 |
| onQuoteSelection | `() => void` | 是 | 引用选中 |

**内容块类型：**
- `text` — 文本行（支持段落、标题、列表、代码）
- `table` — 表格（含可编辑表头和单元格）
- `image` — 图片（含可编辑图例）

**支持的行级格式：**
- 加粗 (bold)
- 斜体 (italic)
- 下划线 (underline)
- 删除线 (strikethrough)
- 高亮 (highlight)
- 对齐方式 (left/center/right)

| Prop | 类型 | 必填 | 说明 |
|------|------|------|------|
| messages | `ChatMessage[]` | 是 | 消息列表 |
| input | `string` | 是 | 输入框内容 |
| quotedText | `string \| null` | 是 | 引用的文本 |
| tokenUsage | `number` | 是 | Token 已用量 |

| onSend | `() => void` | 是 | 发送消息 |
| onClearQuote | `() => void` | 是 | 清除引用 |
| loading | `boolean` | 是 | 加载中状态 |

```tsx
// 使用示例
import ChatPanel from "../components/right/ChatPanel";

<ChatPanel
  messages={messages}
  input={input}
  quotedText={quotedText}
  tokenUsage={tokenUsage}
  onInputChange={handleInputChange}

```

## 三、类型定义

### 3.1 章节相关（`types/section.ts`）

```tsx
interface SectionNode {
  id: string;
  title: string;
  index?: string;
  expanded?: boolean;
  children?: SectionNode[];
}

interface ContentBlock {
  id: string;
  sectionId: string;
  title: string;
  contentType: "text" | "table" | "image";
  lines?: ContentLine[];
  tableData?: { headers: string[]; rows: TableRow[] };
  imageUrl?: string;
  imageCaption?: string;
}

interface ContentLine {
  id: string;
  text: string;
  type: "paragraph" | "heading" | "list" | "code";
  level?: number;
  footnoteRef?: string;
  highlighted?: boolean;
  bold?: boolean;
  italic?: boolean;
  underline?: boolean;
  strikethrough?: boolean;
  align?: "left" | "center" | "right" | "justify";
  color?: string;
}

interface FootnoteReference {
  id: string;
  refNumber: string;
  sourceText: string;
  sourceDesc: string;
}
```

interface ChatMessage {
